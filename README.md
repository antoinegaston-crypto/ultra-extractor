# Ultra-Extractor :

Politique de confidentialité

Ultra Extractor respecte la confidentialité de ses utilisateurs et s’engage à utiliser les autorisations demandées uniquement pour le bon fonctionnement de l’application.

L’application peut demander un accès à vos données et aux disques amovibles afin de permettre l’extraction, la lecture, le transfert ou l’organisation de fichiers depuis différents supports compatibles, comme les clés USB, cartes SD, CD audio, disques durs externes et appareils photo.

Ces accès sont nécessaires uniquement pour permettre à l’utilisateur de sélectionner et traiter ses propres fichiers. Ultra Extractor ne collecte pas automatiquement le contenu de vos fichiers et n’effectue aucune extraction sans action volontaire de votre part.

Aucune donnée personnelle n’est vendue, partagée ou transmise à des tiers.

L’application ne permet pas l’extraction de données depuis les DVD, smartphones ou tablettes.

Le projet et/ou certaines informations liées à l’application peuvent être disponibles sur GitHub, notamment pour présenter l’application, son fonctionnement ou ses mises à jour.

En utilisant Ultra Extractor, vous acceptez que l’application puisse demander les autorisations nécessaires pour accéder à vos données et aux disques amovibles afin d’assurer ses fonctionnalités principales.
