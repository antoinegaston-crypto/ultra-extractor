ultra Extractor ne collecte pas de données personnelles à des fins de suivi publicitaire.

L’application accède uniquement aux fichiers, dossiers, CD, cartes SD, clés USB, disques externes ou appareils sélectionnés par l’utilisateur afin de copier, importer ou partager les fichiers demandés.

L’application peut effectuer des connexions Internet sortantes vers des services de bases de données musicales, comme gnudb.org ou musicbrainz.org, afin de rechercher les titres de pistes audio de CD. Les informations utilisées pour cette recherche sont limitées aux données techniques nécessaires à l’identification du disque ou des pistes.

L’application peut ouvrir l’app Mail de macOS lorsque l’utilisateur choisit d’envoyer un retour depuis la fiche contact. Le message n’est envoyé que si l’utilisateur le valide dans son client Mail.

Les fichiers importés restent sur le Mac de l’utilisateur ou dans le dossier de sortie choisi par l’utilisateur. Ils ne sont pas envoyés vers un serveur appartenant au développeur.
